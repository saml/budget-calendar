# Build

## Prerequisites

- Node.js 24 (matches `.nvmrc`)
- npm

## Setup

```sh
nvm use 24
npm install
```

## Dev server

```sh
nvm use 24
npm run dev
```

Starts Vite dev server at `http://127.0.0.1:5173`.

## Production build

```sh
nvm use 24
npm run build
```

Output in `dist/`.

## Preview production build

```sh
nvm use 24
npm run preview
```
